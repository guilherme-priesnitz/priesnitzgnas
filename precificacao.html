Premissas e fórmulas ▾

Modelo de IVA: no regime regular, o imposto é neutro para a empresa (repasse). A sua margem = base sem imposto. Créditos de insumos aliviam o custo efetivo quando o cliente é consumidor final.

# Hoje
líquido = preço × (1 − alíq_simples)

# Pós-reforma (por fora)
— objetivo “manter margem”:
base = líquido_hoje · preço_cliente = base × (1 + alíq_ref)
— objetivo “manter preço”:
preço_cliente = preço_hoje · base = preço ÷ (1 + alíq_ref)

Regra de bolso: serviços (poucos insumos creditáveis, ISS baixo hoje) tendem a encarecer para o consumidor final; comércio e indústria, com crédito amplo, tendem a ter alívio. Se o seu cliente é empresa (B2B), ele recupera o imposto destacado como crédito — então o preço “por fora” não o assusta.
